ng-form-ui Examples


Use these form elements to spice up your forms and keep things beautiful!


focusMe

Focus an element when the page loads

<input type="text" focus-me/>

slideToggle

Toggle a boolean value using a fancy slider

<slide-toggle ng-model="slideToggleOnOff"><slide-toggle>
ng-model: {{ slideToggleOnOff }}
<slide-toggle onlabel="Active" offlabel="Inactive" ng-model="slideToggleActiveInactve"><slide-toggle>
ng-model: {{ slideToggleActiveInactve }}
